This folder contains sample scripts for using the /asr/meeting/join API to join Webex meeting


In order to run the webex_meeting_join.py script:
1) Replace jwt token with your generated jwt token
2) replace webex meeting link with the invitation link of the ongoing webex meeting(meetingUrl).
3) Start the webex meeting if it's not yet started and run the below command

`python webex_meeting_join.py`
